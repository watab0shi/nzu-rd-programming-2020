## 文字列（String）

シングルクォーテーション `''` またはダブルクォーテーション `""` で囲った中身はプログラム上で文字列として扱われます。  
数値（Number）や真偽値（Boolean）と並ぶデータ型のひとつです。

```javascript
console.log('moji');
console.log("文字");

let moji = 'もじ';
console.log(moji);
```

実行結果

```
moji
文字
もじ
```

### 文字列同士の連結

`+` 演算子を使うことで文字列同士の連結が可能です。  
また、変数を活用することで動的に文字列を作ることもできます。

```javascript
console.log('p5' + 'js');

let name = '猫';
let message = '私は' + name + 'です。';
console.log(messgae);
```

実行結果
```
p5js
私は猫です。
```
